Why have an "About Cale" page?

We can communicate instantly across the world, but it makes it harder to understand who we are talking to. If you have not met me or you do not know me well, I hope the information on this page will lower the barriers for you to form a deeper connection with me. I am sure we have something in common; let's start there!

How did you build this website?

I built every part of this website from scratch. That includes all the JavaScript functionality and CSS styling. I wanted to build a site without using libraries like Bootstrap or Bulma because I thought it would be fun!
